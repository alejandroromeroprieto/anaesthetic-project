# anaesthetic project

This repo stores the code used for our analysis of warming impact from different types of anaesthesia using the FaIR Simple Climate Model.

The jupyter notebooks used to create the plots for the article are in the `notebooks` folder. These notebooks, in turn, use input data taht live under the `data` folder.
